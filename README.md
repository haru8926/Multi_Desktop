# Multi Desktop (WPF 仮想デスクトップ管理アプリ)

Multi Desktop は、Windows 上で「Main」と「Sub」の2つの異なるデスクトップレイアウト（壁紙、ファイル配置、タスクバー設定など）を即座に切り替えるための常駐型アプリケーションです。

## 用途
- **仕事と趣味の分断化**:仕事ではMainデスクトップを使い、自宅にいるときや趣味のときは、Subデスクトップを使い分けるといった分断化をすることができます。
- **趣味バレなどを防ぐ**:「背景を推しなどにしたいけどバレたら恥ずかしい...」、「背景を変えたいけど、その背景がバレたらまずい...」などがあったとしても、誰かが来た瞬間に、`Ctrl+Shift+S`を押せば瞬時にデスクトップを切り替えることができて、他人にバレずにパソコンを有効活用できます。
- **MacOS風のタスクバー**:このアプリには、タスクバーをMacOS風にできる機能を搭載しています。「普通のWindowsに飽きた...」、「MacOSっぽいUIを試してみたい...」というニーズに対応しています。
- **アプリ内BGM機能**:音楽を聞きたいときにChromeや音楽アプリなどをいちいち開いてタスクバーに保持しておく必要はありません。このアプリケーション内には、ミュージック機能を搭載しており、そこからWebView2を使用して、Youtube、AmazonMusic、Spotifyをバックグラウンドで再生することができます(設定からログインが必要です)。-ボタンでバックグラウンド再生、xボタンを使用して完全にWebView2を閉じることもでき、メモリを圧迫しません。

## 🌟 主な機能

- **デスクトップの瞬時切り替え**: `Ctrl + Shift + S` のグローバルホットキー、またはシステムトレイのアイコンから、いつでも環境を切り替えられます。
- **独立したファイル管理**: モードごとにデスクトップのファイルの実体を切り替えることが可能です（パス指定による切り替え）。
- **個別の壁紙・タスクバー設定**: 
  - モードごとに独立した壁紙を指定可能。
  - **通常モード**: デフォルトのWindowsタスクバーを表示。
  - **macOS風 Dockモード**: Windowsネイティブのタスクバーを隠し、 **macOS風 Dock** と **システムメニューバー** を表示します。
- **インテリジェントな macOS風 Dock**:
  - アイコンホバー時にアプリの **リアルタイムプレビュー（スクリーンショット）** を表示。
  - 右クリックメニューも一部実装済み。
- **音楽ストリーミング連携**:
  - メニューバーから YouTube, Amazon Music, Spotify に素早くアクセスできる音楽プレイヤー機能を搭載。
  - ストリーミングアプリケーションはWebView2で動作していて、-ボタンで起動したまま最小化、xボタンでWebView2を終了させることができます。
- **マルチモニター対応**:
  - 全てのモニターのタスクバーを制御可能。
  - マウスカーソルのあるモニターに Dock を自動表示するスマートな移動機能を備えています。

## 🛠 使い方

1. **アプリの起動**: 起動するとシステムトレイに常駐し、メインの設定画面が開きます。
2. **環境設定**: `⚙ 設定` ボタンから、Main と Sub それぞれの「壁紙」「データフォルダ」「タスクバースタイル」を設定します。
3. **切り替え**: メイン画面のボタン、または `Ctrl + Shift + S` キーで環境が瞬時に切り替わります。
4. **Dock の活用**: Dock モードでは、よく使うアプリを右クリックメニューや設定画面から自由に追加・管理できます。
5. **最小化**: `×` ボタンで閉じてもアプリは終了せず、システムトレイに常駐します。完全に終了するには設定画面内の「終了」ボタンを使用してください。

## 💻 動作環境・技術スタック

- **OS**: Windows 10 / 11
- **フレームワーク**: .NET 8.0 WPF
- **言語**: C#

## ⚠️ 注意事項

- 本アプリは Windows API を使用してタスクバーの状態（自動非表示等）を制御しています。万が一タスクバーが戻らなくなった場合は、アプリを再起動して正常に終了させるか、タスクマネージャーから `explorer.exe` を再起動してください。
- このアプリケーションは、Gemini3.1Pro及びClaudeOpus4.6によって作成されています。コード品質やハードウェア使用にはダブルチェックなどを用いて万全を期していますが、万が一、不適切な内容が含まれていたり、アプリの不具合が発生した場合は、ご連絡いただけると幸いです。
