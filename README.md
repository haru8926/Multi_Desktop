# Multi Desktop (WPF 仮想デスクトップ管理アプリ)

Multi Desktop は、Windows 上で「メイン環境」と「サブ環境」の2つの異なるデスクトップレイアウト（壁紙、タスクバー設定、ショートカットなど）を即座に切り替えるための常駐型アプリケーションです。

## 主な機能

- **デスクトップの瞬時切り替え**: `Ctrl + Shift + S` のグローバルホットキーで、いつでも「Main」モードと「Sub」モードをトグルできます。
- **個別の壁紙・フォルダ管理**: モードごとに独立した壁紙と対象データフォルダを指定可能です。
- **タスクバー・Dockスタイル設定**: 
  - **通常モード**: デフォルトのWindowsタスクバーを表示します。
  - **MacOS Dockモード**: Windowsネイティブのタスクバーを隠し、画面下部にmacOS風のアプリランチャー（Dock）、画面上部にmacOS風のシステムメニューバーを表示します。
  - Dockアイコンのホバー時に、対象アプリのリアルタイムプレビュースクリーンショットを表示します。
- **起動中プロセスからの登録**: 設定画面から、お気に入りのアプリや現在起動中のプロセスを選んでDockにピン留めすることができます。
- **誤操作防止**: 設定画面の最小化ボタン無効化や、意図しない終了を防ぐためバックグラウンド（トレイ）への最小化機能などを備えています。完全に終了させたい場合は設定ウィンドウ内の「終了」ボタンを使用します。

## 使い方

1. アプリを起動すると、システムトレイ（タスクバー右下）にアイコンが表示され、設定画面ウィンドウ（MainWindow）が開きます。
2. その画面の `⚙ 設定` ボタンから、各モード（Main / Sub）のフォルダ、壁紙、タスクバスタイル（Dockモード等）を設定します。Dockモードを利用する場合、ここでピン留めしたいアプリも登録します。
3. `🔵 Main に切り替え` または `🟠 Sub に切り替え` ボタンでモードを切り替えます。
4. ウィンドウの `×` ボタンを押すと、アプリは終了せずタスクバーのシステムトレイに入り裏で動き続けます。
5. バックグラウンド動作中であっても、ホットキー `Ctrl + Shift + S` を押すだけでいつでも環境を切り替えられます。

## 動作環境・技術スタック

- **OS**: Windows 10 / 11
- **フレームワーク**: .NET 8.0 WPF
- **言語**: C#

## 補足事項・注意事項

- このアプリは Windows API に強く依存しており、タスクバーの強制的な表示・非表示 (AutoHideの管理等) を行っています。アプリがクラッシュした場合等にWindowsのタスクバーが消えたままになることがありますが、その場合はアプリをもう一度起動して正常に終了させるか、エクスプローラー（`explorer.exe`）を再起動してください。
- マルチモニター環境にも一部対応しています。
- このアプリケーションは、Gemini3.1Pro及びClaudeOpus4.6によって作成されています。コード品質やハードウェア使用にはダブルチェックなどを用いて万全を期していますが、万が一、不適切な内容が含まれていたり、アプリの不具合が発生した場合は、ご連絡いただけると幸いです。
